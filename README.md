# Riqueza Oculta V2

Rebuild unificado e orientado a conversao:

1. Landing
2. Captura de nome
3. Quiz dinamico
4. Resultado com oferta discreta
5. Pagina de vendas
6. Captura WhatsApp
7. Pagamento
8. Pos-pagamento

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Drizzle ORM + PostgreSQL
- Zustand (estado do funil)

## Setup

```bash
cp .env.example .env
npm install
npm run dev
```

Abrir: `http://localhost:3000`

## Nota

Sem variaveis de pagamento configuradas, o projeto opera em modo seguro de simulacao para fluxo de UX.
